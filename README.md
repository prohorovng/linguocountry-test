# Підсумковий тест

Next.js-приложение для студенческого теста:

- студент вводит фамилию/имя и академическую группу;
- проходит тест с таймером на 60 минут;
- результат сохраняется через API;
- таблица результатов доступна по ссылке `/results`;
- CSV-версия доступна по ссылке `/results.csv`.

## Локальный запуск

```powershell
npm install
npm run dev
```

Открыть тест:

```text
http://127.0.0.1:3000
```

Открыть таблицу результатов:

```text
http://127.0.0.1:3000/results
```

## Подготовка к Vercel

Для деплоя нужна постоянная база результатов. Приложение уже поддерживает Upstash Redis через Vercel Marketplace.

1. Создайте проект на Vercel из этого репозитория.
2. В Vercel Dashboard откройте Marketplace и добавьте Upstash Redis к проекту.
3. Vercel автоматически добавит переменные:
   - `UPSTASH_REDIS_REST_URL`
   - `UPSTASH_REDIS_REST_TOKEN`
4. Деплой запускается стандартно: Vercel сам определит Next.js.

Без Upstash Redis приложение тоже работает локально, но на Vercel результаты не должны храниться в файловой системе, потому что serverless-файлы не являются постоянным хранилищем.
